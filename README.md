KOPPEE - A Full-Stack Coffee Shop Web App (MERN)

KOPPEE is a modern and responsive coffee shop web application built with the MERN stack (MongoDB, Express.js, React, and Node.js). Designed for cafes and coffee shops, this app provides a seamless user experience with features such as:

☕ Dynamic Menu Management – Add, update, and display coffee items.
🛒 Online Ordering System – Customers can browse the menu and place orders.
🔒 User Authentication – Secure login and signup with JWT authentication.
📅 Table Reservation – Users can book tables online.
📊 Admin Dashboard – Manage orders, menu items, and reservations.
📩 Newsletter Subscription – Capture customer emails for promotions.
📱 Mobile-Friendly & Responsive – Optimized for all devices.
